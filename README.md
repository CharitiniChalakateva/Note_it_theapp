# Note_it_theapp
Εργασία για το Διαγωνισμό Καινοτομίας και Επιχειρηματικότητας

Η εφαρμογή δημιουργήθηκε αποκλειστικά για τον διαγωνισμό.

Οδηγίες Εκτέλεσης

1. Κάνε clone το repository στο Visual Studio Code:

git clone https://github.com/CharitiniChalakateva/Note_it_theapp.git


2. Άνοιξε το terminal και πήγαινε στο φάκελο του project:

cd Note_it_theapp
ls

Αυτό θα σου εμφανίσει τον φάκελο notedit και το Readme.

3. Μπες στο φάκελο notedit και άνοιξε την εφαρμογή:

cd notedit
open index.html

Η εφαρμογή θα ανοίξει στον προεπιλεγμένο browser σου.
